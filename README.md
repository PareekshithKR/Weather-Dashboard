# Local Weather Dashboard

A weather dashboard built with HTML, CSS, and JavaScript.

## Features

- Geolocation API
- Open-Meteo Weather API
- Temperature, Humidity, Rain Data
- 3-Day Forecast
- Local Storage Caching
- Loading Spinner
- Error Handling

## Technologies

- HTML
- CSS
- JavaScript
- Fetch API
- Open-Meteo API

## Run Locally

Open with VS Code Live Server.
